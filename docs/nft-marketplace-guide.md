# Boom NFT Marketplace Integration Guide

Boom is one of the first NFT platforms on Stacks.

## Contract Architecture

Boom NFTs are SIP-009 compliant non-fungible tokens.

## Marketplace Pattern

1. Seller lists NFT by transferring to marketplace contract
2. Buyer sends STX payment
3. Contract transfers NFT to buyer and STX to seller
